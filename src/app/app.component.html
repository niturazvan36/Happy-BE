Welcome to the Happiness Dashboard


The application is a dashboard for visualizing how a team feels about certain topics. The application provides the possibility to create different polls on which users can vote anonymously. The users are able to submit how they feel about a topic on a configurable scale along with a comment. For each poll, the application provides helpful graphics to analyze the results of the survey.
